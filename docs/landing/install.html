ShipASO is a Claude Code plugin — the ASO reasoning loop runs right in your editor, on your machine, with your credentials. Two commands to install, one to run. No account, no paid ASO API.

Prerequisite: Claude Code — the CLI/editor agent. Don't have it? Get Claude Code →

01

Add the marketplace + install

In Claude Code, paste these two lines. The first registers this repo as a plugin marketplace; the second installs the plugin from it.

Claude Code
> /plugin marketplace add airowe/store-ops > /plugin install store-ops@store-ops

That installs 27 skills — the full audit → research → optimize → ship → verify loop, plus the ship-safely + convert layer (review-risk, rejection assistant, PPO).

02

Run your first audit

Point it at any app by slug or bundle id. This one's read-only — it pulls your live listing and scores every field. No credentials needed to start.

Claude Code
> /aso-audit --app your-app --store appstore
→

03

Where to go next

The audit tells you which skill to run for each gap. The common path:

  • /aso-keyword-research Find terms worth ranking for — expands seeds on real autocomplete + competitor data. No paid API.
  • /aso-metadata-optimization Generate final copy at exact char limits and print the push commands. Never auto-ships.
  • /aso-review-risk Pre-flight for rejections — scans copy for 2.3.7 triggers before you submit.
  • /aso-rank-monitor Prove the rank moved — log your organic rank over time and watch the deltas.
Nothing ships without you. The plugin reasons over free + owned data and prints the exact asc / gplay push commands — you run the push. To execute, install the asc / gplay CLIs and auth with your own developer credentials; the reasoning skills work standalone and emit the commands regardless.
Want it to run itself? The plugin is the whole loop, free, run by you. The hosted agent reruns it weekly — tracking rank, watching competitors, drafting the next optimization, and pinging you only when there's a real move to approve.
